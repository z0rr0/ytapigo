YtapiGo

It is a program to translate and check spelling using the console, it based on Yandex Translate API. By default UTF-8 encoding is used.

It's a clone of the project Ytapi but on the Go programming language. This is created as a package/library, but it can be used as a separate program (see main.go.example), the documentation contains details about all methods and variables.

A spell check is supported only for English, Russian and Ukrainian languages.

Usage

See example in main.go.example.

The latest versions of binary files are available in Releases

Usage:

chmod u+x ytapigo
./ytapigo en-fr Hello dear fried!
# output: Bonjour chers frit!

Dependencies

Standard Go library.

API keys

Users should get API KEYs before an using this program, these values have to be written to a file $HOME/.ytapigo.json (see the example ytapigo_example.json). APIlangs is a set of available translate directions, each one can have a list of possible user's aliases.

{
  "APItr": "some key value",
  "APIdict": "some key value",
  "Aliases": {                      // User's languages aliases
    "en-ru": ["en", "англ", "e"],
    "ru-en": ["ru", "ру" "r"]
  },
  "Default": "en-ru"                // default translation direction
}
  1. APItr - API KEY for Yandex Translate
  2. APIdict - API KEY for Yandex Dictionary

It was implemented using the services:

  • Yandex Dictionary
  • Yandex Translate
  • Yandex Speller
